<h1>About</h1>
The TwitPic class provides full access to the read-only part of the TwitPic API due to browser security restrictions. If you need write access to the API, you may want to check out the <a href="http://github.com/meltingice/TwitPic-API-for-PHP">TwitPic API for PHP</a> project.
